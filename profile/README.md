# FormalQFT

Machine-verified mathematics for quantum field theory and mathematical physics, formalized in [Lean 4](https://lean-lang.org/) with [Mathlib](https://github.com/leanprover-community/mathlib4).

## Repositories

| Repo | Description | Status |
|------|-------------|--------|
| [auto1](https://github.com/FormalQFT/auto1) | Special functions (Gamma, Bessel, orthogonal polynomials) | In progress |
| [stochasticpde-itocalculus](https://github.com/FormalQFT/stochasticpde-itocalculus) | Ito formula and stochastic calculus | Complete |

## Goals

- Build a library of formally verified identities, inequalities, and functional equations for special functions
- Formalize stochastic calculus and stochastic quantization of field theories
- Contribute upstream to Mathlib where possible
